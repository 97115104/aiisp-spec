# Revision history

This file tracks substantive changes to the AIISP family of specifications. Editorial-only changes (typos, formatting) are not recorded here; see git history.

## AIISP-2

### v0.1 - 2026-05-06

Initial AIISP-2 draft for the Human Data Collective v2 proposal.

Added contributor-aligned inference routing and settlement as the active standards-track draft, including:

- `POST /v2/inference`;
- Category Models and contributor-operated Inference Nodes;
- accepted data contribution events;
- verified inference served events or settled inference unit events;
- accepted DPO pair events;
- event-based HDCT issuance with duplicate-event rejection;
- environmental lines for inference;
- public settlement batches;
- Draft v0.1 JSON schemas and examples;
- reference implementation skeleton directories.

AIISP-1 remains in the repository as a historical and provider-facing draft.
